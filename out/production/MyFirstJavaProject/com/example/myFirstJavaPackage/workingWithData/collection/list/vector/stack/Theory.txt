Stack - устаревший synchronized класс, наследует Vector.
Использует принцип LIFO (last in, first out)
Как стопка тарелок: взять можем только самую верхнюю, которую положили.
Чтобы взять вторую с верха, надо снять первую и т.д.

stack.push() - добавить элемент в стэк
stack.pop() - вернуть и удалить элемент из стэка (самый верхний, конечно же)
stack.peek() - просто вернуть элемент из стэка (самый верхний, конечно же)
